   Proiectul consta in implementarea jocului Pong in consola. Utilizatorul poate alege sa joace pe o harta cu obstacole sau alta fara, impotriva calculatorului sau altui jucator, precum si nivelul de dificultate imotriva calculatorului daca este cazul. Paleta din stanga este controlata utilizand tastele w,s ,iar cea din dreapta utilizand i,k, daca este cazul.
Observarie: Proiectul a fost creat utilizand Code Blocks si utilizeaza functia PlaySound(), de aceea in cadrul setarilor compilatorului trebuie adaugat un link catre libraria winmm, daca acest link nu este deja existent.
